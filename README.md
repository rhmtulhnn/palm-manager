## Palm Manager
![Cover](./cover-manager.png)

---

**Palm Manager** is a real-time mobile platform that manages and records harvest, finance, and sales data for PT. Agro Sawit Musi Rawas. It features interactive dashboards, delivers actionable analytics, and generates exportable reports to support efficient, data-driven decision-making in palm oil plantation operations.
